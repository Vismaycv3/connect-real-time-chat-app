# Connect - Real-Time Chat Application

### Live Demo
🔗 https://connect-real-time-chat-app.vercel.app

## Features

- Real-time messaging using Socket.IO
- JWT Authentication
- User Registration & Login
- Online/Offline Status Tracking
- MongoDB Atlas Database
- Responsive UI

## Tech Stack

- React
- Node.js
- Express.js
- MongoDB Atlas
- Socket.IO
- JWT
- Tailwind CSS
